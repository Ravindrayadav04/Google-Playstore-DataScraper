# Google Play Store Scraper

---
## Overview
This project is a **Flask-based web application** that allows users to fetch details about an Android app from the **Google Play Store** using its App ID.  
It uses the `google-play-scraper' Python library to retrieve **app details, reviews, permissions, and related apps**.
Now the project also supports **MongoDB integration**, meaning scraped app data is automatically stored in a local MongoDB database.
